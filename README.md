# wa-auto — WhatsApp bulk sender (fully automated)

## Setup
```bash
# needs Node.js + Chromium
sudo pacman -S nodejs npm chromium

npm install
```

## Usage
1. Edit `send.js` — paste your MESSAGE and 36 NUMBERS
2. `node send.js`
3. Scan QR once with your phone → WhatsApp → Linked Devices
4. All messages send automatically, no clicks needed
